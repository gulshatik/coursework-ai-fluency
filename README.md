# ai-fluency

## План личной AI‑грамотности

> **Ссылка на курс**: [AI Fluency Framework Foundations](https://anthropic.skilljar.com/ai-fluency-framework-foundations)

---

### 1. Введение

AI Fluency Framework Foundations – это структурированный подход к развитию навыков работы с искусственным интеллектом. Он помогает понять, как ИИ меняет бизнес, как правильно формулировать задачи, оценивать модели и внедрять решения. В этом документе описан персональный план, который позволит мне пройти курс, выполнить финальное задание и закрепить знания на практике.

---

### 2. Цели

| Период | Цель | Метрика успеха |
|--------|------|----------------|
| **Краткосрочная (0‑3 мес.)** | Завершить курс и получить сертификат | Сертификат + 5/5 оценка в финальном тесте |
| **Среднесрочная (3‑12 мес.)** | Создать и развернуть простое приложение на основе LLM | Приложение работает, пользовательский тест ≥ 80 % удовлетворённости |
| **Долгосрочная (12+ мес.)** | Внедрить AI‑решение в рабочий процесс компании | Снижение времени выполнения задачи на 30 % и экономия затрат ≥ 15 % |

---

### 3. Путь обучения

| Этап | Тема | Подтемы | Формат |
|------|------|---------|--------|
| **1. Основы данных** | Сбор и подготовка данных | Источники, очистка, аугментация | Онлайн‑курс + практические задания |
| **2. Модели и алгоритмы** | Базовые модели LLM | Архитектура, обучение, fine‑tuning | Лаборатория + GitHub репозиторий |
| **3. Этические и правовые аспекты** | Bias, privacy, explainability | Кодекс этики, GDPR | Чтение статей + дискуссии |
| **4. Разработка и деплой** | Инфраструктура, CI/CD | Docker, Kubernetes, API | Практика в облаке |
| **5. Интеграция в бизнес** | Use‑case mapping, ROI | Анализ кейсов, метрики | Проектный кейс |

### 5. Вехи (Milestones)

| Дата | Веха | Описание | Deliverable |
|------|------|----------|-------------|
| **Месяц 1** | Завершить курс | Пройти все модули, пройти финальный тест | Сертификат |
| **Месяц 2** | Создать чат‑бота | Использовать LLM, Docker, API | GitHub репозиторий + работающий бот |
| **Месяц 4** | Провести A/B‑тест | Оценить эффективность бота | Отчёт с метриками |
| **Месяц 6** | Интеграция в бизнес | Внедрить решение в рабочий процесс | Документация и презентация |
| **Месяц 12** | Оценка ROI | Сравнить показатели до/после | Финальный отчёт |

### 6. Рефлексия

- **Что получилось?**  
  Оценить выполненные задачи и достигнутые результаты.  
- **Что не удалось?**  
  Выявить препятствия и причины.  
- **Как улучшить?**  
  Пересмотреть план, добавить новые ресурсы или изменить сроки.  

**Периодичность**: ежемесячно в конце каждого месяца.
